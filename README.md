# 🎂 Zaya's Golden Birthday Wishes

موقع تفاعلي لطيف لتهنئة **Zaya** بعيد ميلادها الذهبي، يتيح للزوار إرسال الأمنيات والرسائل مع تأثيرات بصرية مميزة.

## 🌟 الميزات (Features)

- 🎈 تصميم دافئ بأسلوب أنيق وبسيط.
- ❤️ خلفية تتساقط منها القلوب التفاعلية باستخدام JavaScript.
- 💌 نموذج لإرسال تهاني عيد الميلاد.
- 🗄️ ربط مباشر مع قاعدة بيانات **Supabase** لحفظ الرسائل فوراً.
- 🚀 مستضاف على منصة **Netlify**.

## 🛠️ التقنيات المستخدمة (Tech Stack)

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend / Database:** [Supabase](https://supabase.com)
- **Deployment:** [Netlify](https://www.netlify.com)
